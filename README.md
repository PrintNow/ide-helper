IDE Helper for Hyperf

## Installation

~~~bash
composer require friendsofhyperf/ide-helper
~~~

## Usage

- Generate model helper

~~~bash
php ./bin/hyperf.php ide-helper:model
~~~

- Generate macro helper

~~~bash
php ./bin/hyperf.php ide-helper:macro
~~~
